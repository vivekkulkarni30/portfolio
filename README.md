# VS Code-Inspired Portfolio

A self-contained static portfolio that turns the homepage into an interactive developer workspace.

## Files

- `index.html` contains the semantic shell and metadata.
- `styles.css` defines the VS Code-inspired visual system and responsive layout.
- `data.js` stores portfolio content in one editable data object.
- `app.js` renders the portfolio sections and powers interactions.
- `assets/workspace-mark.svg` is the local hero visual asset.

## Run

Open `index.html` directly in a browser, or serve the folder with any static server.
